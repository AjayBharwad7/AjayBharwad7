## Featured Products

### DevConnect

A production-deployed developer network and project collaboration platform.

- Professional developer discovery and profile search
- Connection requests, approvals, notifications, and private messaging
- Project publishing, bidding, and developer matching
- Skill verification with timed MCQs, coding challenges, badges, and certificates
- Role-based dashboards for developers, teams, and administrators
- Secure JWT authentication backed by managed PostgreSQL

**[Open the live platform](https://devconnect-platform-live.vercel.app)** ·
**[View the code-free showcase](https://github.com/AjayBharwad7/devconnect-showcase)**

An AI-powered project memory platform that connects repositories, documents,

conversations.
